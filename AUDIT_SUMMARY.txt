DEPLOYMENT INSTRUCTIONS
================================================================================

Quick Start:
  1. cd backend
  2. npm install (installs bcrypt, jsonwebtoken, dotenv)
  3. cp .env.example .env
  4. Generate bcrypt password hash:
     node -e "require('bcrypt').hash('your-password', 10, console.log)"
  5. Generate JWT secret: openssl rand -hex 32
  6. Set credentials in .env
  7. Configure SMTP (Gmail app password recommended)
  8. Test locally: npm start
  9. Push to Vercel with env vars set in dashboard
  10. Verify admin login works at /admin

Critical Environment Variables:
  ADMIN_USER=admin
  ADMIN_PASS_HASH=$2b$10$... (bcrypt hash)
  JWT_SECRET=a1b2c3d4... (64 hex chars)
  SMTP_HOST=smtp.gmail.com
  SMTP_PORT=587
  SMTP_USER=your-email@gmail.com
  SMTP_PASS=app-password
  ADMIN_EMAIL=admin@talbifluid.fr

See DEPLOYMENT.md for detailed instructions.
